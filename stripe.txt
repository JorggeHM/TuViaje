  Para activar (pasos manuales requeridos):

  1. Correr la migración en MySQL:
  source viaje.database/migration_stripe_sessions.sql;
  2. Crear cuenta Stripe (gratis) y obtener claves de Test en https://dashboard.stripe.com/test/apikeys.
  3. Editar viaje.api/.env:
  STRIPE_SECRET_KEY=sk_test_...
  STRIPE_WEBHOOK_SECRET=  (lo configurás en el paso 4)
  APP_URL=http://localhost:5173
  4. Para recibir webhooks en localhost durante desarrollo, instalar Stripe CLI (https://stripe.com/docs/stripe-cli) y correr:
  ! stripe login
  ! stripe listen --forward-to http://localhost/AgenciaDeViajesSystem/viaje.api/api/stripe/webhook
  4. El comando imprime un whsec_... → ese es el STRIPE_WEBHOOK_SECRET. Mantener corriendo mientras se prueba.
  5. Tarjeta de test para pagar: 4242 4242 4242 4242, cualquier CVC, fecha futura.
